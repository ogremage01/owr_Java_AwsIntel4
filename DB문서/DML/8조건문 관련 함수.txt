8조건문 관련 함수.txt20240612


SWITCH CASE문과 같은 기능을 한다

조건문이다

형식
DECODE(표현식 조건1, 결과1,
			조건2, 결과2,
			조건3, 결과3,
			기본결과N(ELSE와 비슷)
)


*CASE 함수
형식 -> 표현식, EXPRESSION
CASE 	WHEN 조건1 THEN 결과1
		WHEN 조건2 THEN 결과2
		WHEN 조건3 THEN 결과3
		ELSE 결과 N
END


CASE함수는 다양한 비교 연산자를 이용해서 조건을 제시할 수 있기 때문에
범위를 지정할 수도 있다
프로그램 언어의 IF ELSE IF ELSE문과 유사한 구조를 갖는다


DECODE와 CASE ?
DECODE는 오라클 전용 함수이다
조건의 수가 많으면 가독성이 떨어진다

CASE는 SQL 표준으로 다양한 DBMS에서 사용가능하다
조건을 보다 유연하게 사용가능하다
가독성이 좋다
