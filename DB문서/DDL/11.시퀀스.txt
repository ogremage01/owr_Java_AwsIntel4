11.시퀀스.txt

오라클에서는 행을 구분하기 위해 기본키를 두고 있다

기본키는 중복된 값을 가질 수 없으므로 항상 유일한 값을 가져야 합니다.
기본키가 유일한 값을 갖도록 사용자가 직접 값을 다루려면 부담이 되는데
시퀀스는 테이블 내의 유일한 숫자를 자동으로 생성하는 
자동 번호 발생 객체이므로 시퀀스를 기본키 사용시 활용하면 부담을 줄 일 수 있습니다.


표현식
create sequence 시퀀스명
start with n			시작값
increment by n			증가값
maxvalue n				최대값
minvalue n;				최소값

등의 옵션이 있다
